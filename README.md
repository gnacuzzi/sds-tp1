# TP1 — Neighbor Detection (CIM vs Brute Force)

This project implements neighbor detection between particles using two methods:

* **Brute Force**
* **Cell Index Method (CIM)**

The simulation and neighbor calculations are implemented in **C**, while **Python is used for visualization and performance analysis**.

---

.

---

# Compile the Program

From the **root of the project**, run:

```bash
make
```

This creates the executable:

```
tp1
```

---

# Run a Single Simulation

Format:

```bash
./tp1 [N] [M] [periodic]
```

Parameters:

| Parameter  | Meaning                                         |
| ---------- | ----------------------------------------------- |
| `N`        | number of particles                             |
| `M`        | number of cells per dimension                   |
| `periodic` | `0` = walls, `1` = periodic boundary conditions |

Example:

```bash
./tp1 500 10 0
```

This generates the following files inside `data/`:

```
static.txt
dynamic.txt
neighbors_bruteforce.txt
neighbors_cim.txt
```

The program also prints:

* execution time of each algorithm
* whether the neighbor lists match

---

# Visualize the System

The Python visualizer reads the files generated in `data/` and produces an image.

Run:

```bash
python3 python/visualizer.py --target 333 --M 10
```

Parameters:

| Parameter    | Meaning                        |
| ------------ | ------------------------------ |
| `--target`   | particle ID to highlight       |
| `--M`        | number of CIM grid cells       |
| `--periodic` | enables periodic visualization |

Example with periodic boundaries:

```bash
python3 python/visualizer.py --target 333 --M 10 --periodic
```

The image is saved in:

```
output/neighbors_plot.png
```

Visualization colors:

* 🔴 red → target particle
* 🟢 green → neighbors
* 🔵 light blue → other particles
* 🟠 orange circle → interaction radius

# Performance Experiments

To evaluate the efficiency of the algorithms (as required in the assignment), the program can run a set of experiments that vary:

* **N (number of particles)**
* **M (number of grid cells)**

Each configuration is executed multiple times and the execution time of **Brute Force** and **CIM** is recorded.

Run the experiment mode:

```bash
./tp1 experiment
```

This will generate:

```
data/performance.csv
```

The CSV file contains all runs in the format:

```
study,N,M,method,run,time_seconds
```

Example:

```
vary_N,100,10,brute_force,1,0.0012
vary_N,100,10,cim,1,0.00015
```

---

# Generate Performance Plots

After running the experiment, generate the plots with:

```bash
python3 python/performance.py
```

This produces:

```
output/experiment_vary_N.png
output/experiment_vary_M.png
```

The plots show:

* average execution time
* standard deviation (error bars)
* comparison between **Brute Force** and **CIM**
