# TP1 — Neighbor Detection (CIM vs Brute Force)

This project implements neighbor detection between particles using two methods:

* **Brute Force**
* **Cell Index Method (CIM)**

The simulation and neighbor calculations are implemented in **C**, while **Python is used only for visualization**.

---

# Project Structure

```
.
├── data/           # Files generated by the C program
│   ├── static.txt
│   ├── dynamic.txt
│   ├── neighbors_bruteforce.txt
│   └── neighbors_cim.txt
│
├── output/         # Images generated by the Python visualizer
│
├── python/
│   └── visualizer.py
│
├── src/            # C source code
│
└── Makefile
```

---

# Compile the Program

From the **root of the project**, run:

```bash
make
```

This will create the executable:

```
tp1
```

---

# Run the Simulation

Format:

```bash
./tp1 [N] [M] [periodic]
```

Parameters:

| Parameter  | Meaning                                         |
| ---------- | ----------------------------------------------- |
| `N`        | number of particles                             |
| `M`        | number of cells per dimension                   |
| `periodic` | `0` = walls, `1` = periodic boundary conditions |

Example:

```bash
./tp1 500 10 0
```

This generates the following files inside the `data/` folder:

```
static.txt
dynamic.txt
neighbors_bruteforce.txt
neighbors_cim.txt
```

The program also prints:

* execution time for each algorithm
* whether the neighbor lists match

---

# Visualize the System

The Python visualizer reads the files generated in `data/` and produces an image.

Run:

```bash
python3 python/visualizer.py --target 333 --M 10
```

Parameters:

| Parameter    | Meaning                        |
| ------------ | ------------------------------ |
| `--target`   | particle ID to highlight       |
| `--M`        | number of CIM grid cells       |
| `--periodic` | enables periodic visualization |

Example with periodic boundaries:

```bash
python3 python/visualizer.py --target 333 --M 10 --periodic
```

The image is saved in:

```
output/neighbors_plot.png
```

Visualization colors:

* 🔴 red → target particle
* 🟢 green → neighbors
* 🔵 light blue → other particles
* 🟠 orange circle → interaction radius
